📌 **Lesson: Installing the Rust-Based Sui SDK (`sui-sdk-rs`)**

### **1️⃣ Install Rust-Based Sui SDK**
```sh
cargo install sui-sdk
```

### **2️⃣ Verify Installation**
```sh
cargo --version
sui --version
```
✅ If everything is installed correctly, you should see version numbers.
